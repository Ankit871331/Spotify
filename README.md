# Spotify Clone 🎵

A responsive web application that replicates the Spotify user interface, allowing users to browse and interact with music playlists. This project focuses on clean UI/UX design using modern web technologies.

## 🚀 Features
* **Dynamic Sidebar**: Navigation for Home, Search, and Library.
* **Music Player UI**: Bottom-fixed play bar with controls (Play, Pause, Volume, Seek).
* **Responsive Layout**: Designed to work across different screen sizes.
* **Playlist Cards**: Interactive hover effects for album and playlist discovery.

## 🛠️ Tech Stack
* **HTML5**: For structured content.
* **CSS3**: For custom styling, Flexbox, and Grid layouts.
* **JavaScript**: For interactive playback logic like play push.

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Ankit871331/Spotify.git
   ```
2. **Navigate to the directory**:
   ```bash
   cd Spotify
   ```
3. **Open the project**:
   Simply open `index.html` in your favorite web browser.
